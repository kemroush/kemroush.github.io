## DESIGN SPECIFICATIONS

### Color Scheme Suggestions
- **Primary**: Soft pastels (peachy/coral tones for autumn feel)
- **Secondary**: White/cream backgrounds
- **Accent**: Green for natural ingredients
- **Text**: Dark gray (#333333) for readability

### Typography
- **H1**: 36-42px, Bold, Serif (elegant)
- **H2**: 28-32px, Semi-bold
- **H3**: 20-24px, Semi-bold
- **Body**: 16-18px, Regular, Sans-serif
- **Product codes**: 14px, Monospace font

### Spacing
- Section padding: 60-80px vertical
- Card spacing: 24px between cards
- Content max-width: 800px (for readability)
- Sidebar width (if used): 300px

### Ingredient Card Components
```
┌──────────────────────────────────┐
│ ┌────┐                          │
│ │Icon│ INGREDIENT NAME           │  ← 24px heading
│ └────┘                           │
│                                   │
│ Description text explaining the   │  ← 16px body text
│ benefits and usage...             │
│                                   │
│ ┌─────────────────────────┐     │
│ │ Kód: INS0044            │     │  ← 14px, gray bg
│ └─────────────────────────┘     │
│                                   │
│ [Zobrazit produkt →]            │  ← CTA button
└──────────────────────────────────┘
```

## RESPONSIVE BEHAVIOR

### Desktop (1200px+)
- Two-column layout for ingredient cards (side-by-side)
- Full-width hero image
- Floating table of contents (sticky sidebar)

### Tablet (768px - 1199px)
- Single column ingredient cards
- Stacked routine cards
- Reduced padding

### Mobile (< 768px)
- Single column all content
- Collapsed table of contents (expandable)
- Smaller hero image (optimized)
- Touch-friendly buttons (min 44px height)
- Product codes stack below description

### Buttons
- **Primary CTA**: "Zobrazit produkt" (Green, rounded)

- Hover states: Slight scale (1.05) + color darkening

### Product Code Badges
- Copyable on click
- Tooltip: "Klikněte pro kopírování"
- Visual feedback when copied

### Scroll Animations (Optional)
- Fade-in cards as user scrolls
- Progress bar showing reading progress
